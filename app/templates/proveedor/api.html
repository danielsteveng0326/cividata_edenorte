Información sobre la Sincronización

Esta función sincroniza proveedores desde el RUP (Registro Único de Proveedores) del SECOP utilizando la API de datos.gov.co.

¿Qué hace la sincronización?
  • Consulta proveedores activos desde la API oficial
  • Crea registros nuevos si no existen en el sistema
  • Actualiza información existente con datos más recientes
  • Mantiene un log de errores para revisión
Tiempo de Procesamiento

La sincronización puede tomar varios minutos dependiendo de la cantidad de registros. Por favor ten paciencia durante el proceso.
